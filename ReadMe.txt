Shopify_Website
• 	Project is built in Django Framework.
•	First create folder shopify_1 and create app inventory inside the folder.
•	All functions of CURD Operation are coded in the views.py file.
•	Provide the URL of all fields in url.py.
•	Created templates in HTML to design a webpage.
•	Designed templates for CURD operations.
•	Designed JSON code in file forms.py.
•	Defined all fields with their data type in models.py.
•	 For execution run the python manage.py command on the terminal.
•	Once the command is successfully executed open the terminal and type the IP address http://127.0.0.1:8000.
•	For admin page http://127.0.0.1:8000/admin.
•	For JSON format http://127.0.0.1:8000/json.
